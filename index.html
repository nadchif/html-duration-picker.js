Example Usage

Include html-duration-picker.min.js in your HTML file, just before the </body> tag. like this: 

        
        <script src="html-duration-picker.min.js"></script>
        </body>

add a html-duration-picker class on any <input>box. like this: 

        
            <input class="html-duration-picker">

Result:

Regular Input box
<input type="text">


Duration Input box
<input class="html-duration-picker">

Features:

Set default value to 15 minutes <input type="text" data-duration="00:15:00" >

Set max value to 35 minutes <input type="text" data-duration-max="00:35:00" >

Set min value to 5 minutes <input type="text" data-duration-min="00:05:00" >
